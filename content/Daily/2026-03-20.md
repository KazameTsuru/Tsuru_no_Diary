### The RPA theory of a Coulomb Gas (cite from Nagoase QFT for CMP)

Start with the following Coulomb Hamiltonian :
$$\hat{H}=\sum_i\frac{\hat{\vec{p}}_i^2}{2m}+\sum_{i<j}\frac{e^2}{|\hat{\vec{r}}_i-\hat{\vec{r}}_j|}$$

We want to use momentum representation to realize second quantization, thus consider first the density operator :
$$\hat{\rho}(\vec{r})=\sum_{i=1}^N\delta(\vec{r}-\vec{r}_i)\Longrightarrow \hat{\rho}(\vec{k})=\sum_{i=1}^N e^ {-i\vec{k}\cdot\hat{\vec{r}}_i}$$

So we can write our interaction term in the form of density operator as :
$$\sum_{i<j}\frac{e^2}{|\hat{\vec{r}}_i-\hat{\vec{r}}_j|}=\frac{1}{2}\sum_{i,j}\sum_{\vec{q}\neq }\frac{4\pi e^2}{|\vec{q}|^2}e^{i\vec{q}\cdot(\hat{\vec{r}}_i-\hat{\vec{r}}_j)}=\frac{1}{V}\sum_{\vec{q}\neq }\frac{2\pi e^2}{|\vec{q}|^2}\{\sum_j e^{-i\vec{q}\cdot\hat{\vec{r}}_j}\sum_i e^{-i (-\vec{q})\cdot\hat{\vec{r}}_i}-\frac{N}{2}\}$$

Therefore, we can easily use canonical quantization language to obtain the Hamiltonian :
$$H=\int\sum_\sigma\hat{\psi}_\sigma^\dagger(\vec{r})[-\frac{\hbar^2\nabla^2}{2m}-\mu]\hat{\psi}_\sigma(\vec{r})d\vec{r}+\frac{1}{V}\sum_{\vec{q}\neq0}\frac{2\pi e^2}{|\vec{q}|^2}\{\hat{\rho}(\vec{q})\hat{\rho}(-\vec{q})-N\}=\sum_{\vec{k},\sigma}(\frac{\hbar^2\vec{k}^2}{2m}-\mu)\hat{c}^\dagger_{\vec{k}\sigma}\hat{c}_{\vec{k}\sigma}+\frac{1}{V}\sum_{\vec{q}\neq0}\frac{2\pi e^2}{|\vec{q}|^2}\{\hat{\rho}(\vec{q})\hat{\rho}(-\vec{q})-N\}$$

Following derivation of fermion path integral, the action is given by :
$$S=\int_0^\beta d\tau\{\int d\vec{r}\sum_\sigma\bar{\psi}_\sigma(\vec{r},\tau)[\partial_\tau-\frac{1}{2m}\nabla^2 -\mu]\psi_\sigma(\vec{r},\tau)+\frac{1}{V}\sum_{\vec{q}\neq0}\frac{2\pi e^2}{|\vec{q}|^2}\{\rho(\vec{q})\rho(-\vec{q})-N\}\}$$
