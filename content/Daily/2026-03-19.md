### Neural Network based Variational Monte Carlo Tutorial (Netket Tutorial Referred)

#### 1. The Physical System: 1D AFM Heisenberg Chain
Example: Ground state for 1D Anti-ferromagnetic Heisenberg Chain

The Hamiltonian in the tutorial is defined using Pauli matrices sigma:

$$H = \sum_{i=1}^{L} \vec{\sigma}_i \cdot \vec{\sigma}_{i+1}$$

Note that sigma represents the vector of Pauli matrices $(\sigma_x, \sigma_y, \sigma_z)$. There is a factor of 4 difference between this representation and the spin-1/2 operator S, where $S = \sigma / 2$.

##### 2.2 Ansatz/Network Design
First for traditional VMC, we set ansatz like Jastrow with small amount of parameters :
note in netket usage, we use flax instead of commonly seen pytorch/tensorflow to define our network. However, such framework can be translated from each easily.
```python
from flax import nnx
import jax.numpy as jnp
import jax


class Jastrow(nnx.Module):
    def __init__(self, N: int, *, rngs: nnx.Rngs):
        k1, k2 = jax.random.split(rngs.params())
        self.J = nnx.Param(0.01 * jax.random.normal(k1, (N, N), dtype=jnp.complex128))

        self.v_bias = nnx.Param(
            0.01 * jax.random.normal(k2, (N, 1), dtype=jnp.complex128)
        )

    def __call__(self, x):
        x = x.astype(jnp.complex128)  # keep the dtypes aligned
        quad = jnp.einsum("...i,ij,...j->...", x, self.J, x)
        lin = jnp.squeeze(x @ self.v_bias, -1)  # (...,N) @ (N,1) → (...,1)
        return quad + lin
```

##### 3.1 Generate a cutomized lattice or hamiltonian

Here we use classical J1-J2 Model as example to show how manually we define a lattice and hamiltonian
```python
# Define custom graph
edge_colors = []
for i in range(L):
    edge_colors.append([i, (i + 1) % L, 1])
    edge_colors.append([i, (i + 2) % L, 2])

# Define the netket graph object
g = nk.graph.Graph(edges=edge_colors)

# Sigma^z*Sigma^z interactions
sigmaz = [[1, 0], [0, -1]]
mszsz = np.kron(sigmaz, sigmaz)

# Exchange interactions
exchange = np.asarray([[0, 0, 0, 0], [0, 0, 2, 0], [0, 2, 0, 0], [0, 0, 0, 0]])

bond_operator = [
    (J[0] * mszsz).tolist(),
    (J[1] * mszsz).tolist(),
    (-J[0] * exchange).tolist(),
    (J[1] * exchange).tolist(),
]

bond_color = [1, 2, 1, 2]

# Spin based Hilbert Space
hi = nk.hilbert.Spin(s=0.5, total_sz=0.0, N=g.n_nodes)

# Custom Hamiltonian operator
op = nk.operator.GraphOperator(
    hi, graph=g, bond_ops=bond_operator, bond_ops_colors=bond_color
)
```
##### 3.2 Design the Network
```python
class DetailedRBM(nn.Module):
    alpha: int = 1

    @nn.compact
    def __call__(self, x):
        x_hidden = nn.Dense(
            features=self.alpha * x.shape[-1],
            use_bias=True,                         
            bias_init=nn.initializers.normal(stddev=0.1) 
        )(x)
        
        h = jnp.log(jnp.cosh(x_hidden)).sum(axis=-1)

        v_bias = self.param("v_bias", nn.initializers.zeros, (x.shape[-1],))
        
        return h + jnp.dot(x, v_bias)
```

##### 3.4 DIY Optimization
```python
from tqdm import tqdm

energy_history = []
n_steps = 100

# For every iteration (tqdm is just a progress bar)
for i in tqdm(range(n_steps)):
    # compute energy and gradient of the energy
    E, E_grad = vstate.expect_and_grad(H)
    # log the energy to a list
    energy_history.append(E.mean.real)
    # equivalent to vstate.parameters - 0.05*E_grad , but it performs this
    # function on every leaf of the dictionaries containing the set of parameters
    new_pars = jax.tree_util.tree_map(
        lambda x, y: x - 0.05 * y, vstate.parameters, E_grad
    )
    # actually update the parameters
    vstate.parameters = new_pars
```
